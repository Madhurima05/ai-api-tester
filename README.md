# AI API Tester Pro

A full-stack AI-powered web application testing tool that combines API testing, cross-browser UI testing with screenshots, edge case detection and AI analysis.

## Live Demo
https://ai-api-tester-production.up.railway.app

## Features
- API testing with AI-generated test cases
- Cross-browser testing — Chrome, Firefox and Safari
- Real screenshots of the actual webpage
- Edge case detection automatically
- AI analysis of all test results
- Test history tracking
- Works on any public website or API

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Browser Testing: Playwright
- AI: Groq AI (LLaMA 3)
- Java Version: RestAssured + JUnit 5
- Deployment: Railway + Docker
- CI/CD: GitHub Actions

## How to Run Locally
```bash
cd javascript-project
npm install
node server.js
```
Open http://localhost:3000
