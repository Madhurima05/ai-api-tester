# AI-Powered API Test Automation Framework

An intelligent API testing framework that uses Groq AI to automatically generate test cases and analyze results.

## Tech Stack
- JavaScript: Axios + Jest + Groq AI
- Java: RestAssured + JUnit 5

## Features
- Auto-generates API test cases using AI
- Runs automated tests against live APIs
- AI analyzes test results and suggests improvements
- Supports both JavaScript and Java

## How to Run

JavaScript:
cd javascript-project
npm install
node tests/aiGenerator.js

Java:
cd java-project
mvn test
